**Open-source software** (**OSS**) is [computer software](https://en.wikipedia.org/wiki/Software "Software") that is released under a [license](https://en.wikipedia.org/wiki/Open-source_license "Open-source license") in which the [copyright](https://en.wikipedia.org/wiki/Copyright "Copyright") holder grants users the rights to use, study, change, and [distribute the software](https://en.wikipedia.org/wiki/Software_distribution "Software distribution") and its [source code](https://en.wikipedia.org/wiki/Source_code "Source code") to anyone and for any purpose.[[1]](https://en.wikipedia.org/wiki/Open-source_software#cite_note-1)[[2]](https://en.wikipedia.org/wiki/Open-source_software#cite_note-2) Open-source software may be developed in a collaborative, public manner. Open-source software is a prominent example of [open collaboration](https://en.wikipedia.org/wiki/Open_collaboration "Open collaboration"), meaning any capable user is able to [participate online](https://en.wikipedia.org/wiki/Online_collaboration "Online collaboration") in development, making the number of possible contributors indefinite. The ability to examine the code facilitates public trust in the software.[[3]](https://en.wikipedia.org/wiki/Open-source_software#cite_note-Open_Collaboration-3)
