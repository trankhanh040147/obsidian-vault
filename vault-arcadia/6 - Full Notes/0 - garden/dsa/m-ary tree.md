**Wiki: https://en.wikipedia.org/wiki/M-ary_tree**
- In [graph theory](https://en.wikipedia.org/wiki/Graph_theory "Graph theory"), an **_m_-ary tree** (for nonnegative integers _m_) (also known as **_n_-ary**, **_k_-ary**, **_k_-way** or **generic** tree) is an [arborescence](https://en.wikipedia.org/wiki/Arborescence_\(graph_theory\) "Arborescence (graph theory)") (or, for some authors, an [ordered tree](https://en.wikipedia.org/wiki/Ordered_tree "Ordered tree"))[[1]](https://en.wikipedia.org/wiki/M-ary_tree#cite_note-1)[[2]](https://en.wikipedia.org/wiki/M-ary_tree#cite_note-2) in which each node has no more than _m_ children. A [binary tree](https://en.wikipedia.org/wiki/Binary_tree "Binary tree") is an important case where _m_ = 2; similarly, a [ternary tree](https://en.wikipedia.org/wiki/Ternary_tree "Ternary tree") is one where _m_ = 3.
